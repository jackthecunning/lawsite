# Attorney Images Directory

This directory contains attorney profile photos for the law firm website.

## File Naming Convention
Attorney images should be named using the format: `firstname-lastname.jpg`

Examples:
- `john-smith.jpg`
- `sarah-johnson.jpg`
- `michael-brown.jpg`

## Supported Formats
- `.jpg` / `.jpeg`
- `.png`
- `.webp`
- `.gif`

## Image Requirements
- **Recommended size**: 400x400 pixels (square aspect ratio)
- **Minimum size**: 300x300 pixels
- **File size**: Keep under 500KB for optimal loading

## Fallback System
If an attorney's image is not found, the system will automatically display a professional grey silhouette placeholder.

## Privacy Notice
Actual attorney images are not stored in the repository for privacy reasons. Place your attorney photos in this directory locally.
