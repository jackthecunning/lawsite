# Banner Images Directory

This directory contains hero/banner images for the law firm website.

## File Naming Convention
Banner images are referenced in the Hero component:
- `philly_1.png`
- `philly_2.png`
- `philly_3.png`

## Supported Formats
- `.jpg` / `.jpeg`
- `.png`
- `.webp`
- `.gif`

## Image Requirements
- **Recommended size**: 1920x1080 pixels (16:9 aspect ratio)
- **Minimum size**: 1200x600 pixels
- **File size**: Keep under 2MB for optimal loading
- **Content**: Professional images representing the firm or city

## Fallback System
If banner images are not found, the system will automatically display a professional grey city skyline silhouette.

## Usage
Banner images rotate in a slideshow on the homepage hero section with a 5-second interval.

## Privacy Notice
Actual banner images are not stored in the repository. Place your banner photos in this directory locally.
